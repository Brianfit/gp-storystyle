Activity

Stop, Start, Steroids

Breaking participants into 7 teams – one for each of the 7 shifts, this is a quick exercise to look at what it would mean if we really lived each shift.

Made by Lucy, Laura and friends at Greenpeace.

Steps for the Activity

  1. Break the group into 7 small teams and assign each team one of the 7 shifts.
  2. 5 mins - brainstorm all the things you would need to STOP doing if you were to really live this shift
  3. 5 mins - brainstorm all the things you would need to START doing if you were to really live this shift
  4. 5 mins – brainstorm all the things you would need to AMPLIFY if you were to really live this shift
  5. 5 mins – going back over your lists agree one thing from each to present back to the rest of the team – your presentation should be no more than 1 piece of flipchart paper.

Materials

  • Markers
  • Post-its
  • Flips charts

Discussion

Focus the discussion on the behaviors that participants need to start and stop. Discuss ways people can create habits out of the new behaviors. For example, if participants want to STOP publishing internally, how would thei work flow need to change to prepare information for a wider audience? Reflection is an important part of the learning process, so be sure and discuss the activity!
